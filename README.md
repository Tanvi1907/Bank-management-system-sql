# Bank Management System (SQL)

This is a SQL project for managing bank customers and accounts.

## Features
- Customer management
- Account details
- SQL joins and queries

## Technology
- MySQL
- SQL

## Author
Tanvi Rastogi

## Advanced Features
1. Stored Procedures:
   - sp_deposit(account_id, amount)
   - sp_withdraw(account_id, amount)
   - sp_transfer(from_account, to_account, amount)

2. Triggers:
   - Auto prevent negative balance
   - Auto log transactions in audit_log

3. Views:
   - high_balance_customers
   - monthly_transactions

4. Indexes:
   - Faster query on accounts and transactions

5. Audit Log:
   - Table audit_log tracks all operations
